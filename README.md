# NYC Central Park Squirrel Tracker

Shiny R app for the 2018 Central Park squirrel census.

![Banner](img/squirrels_image.png)

App Dashboard: https://artazyan-nyc-centralpark-squirrel.share.connect.posit.cloud

## Setup

```r
install.packages(readLines("requirements.txt"))
```

## Run locally

```r
shiny::runApp(".")
```
